# og

An inverted [The Go Gopher (gopher)](https://go.dev/blog/gopher) falls from the sky.

![](./og.gif)

## Install

```bash
go install github.com/hamao0820/og@latest
```

### Mac

```bash
brew install hamao0820/tap/og
```

## Usage

```bash
og
```
