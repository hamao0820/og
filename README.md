# og

An inverted [The Go Gopher (gopher)](https://go.dev/blog/gopher) falls from the sky.

![](./og.gif)

## Install

```bash
$ git clone https://github.com/hamao0820/og.git
$ cd og
$ CGO_ENABLED=1 go build ./...
```

## Usage

```bash
og
```
